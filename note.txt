PROJECT NAME: Campus Issue Reporter (Frontend Only)

GOAL:
Build a modern, responsive, light-themed web application where users can upload images of campus issues (like cracks, broken tiles, etc.), add structured details, and view all reported issues.

TECH STACK:

* React 19 + TypeScript (Vite)
* TailwindCSS
* shadcn/ui components
* React Router DOM
* Framer Motion (for smooth animations)
* Lucide Icons

IMPORTANT:

* DO NOT change existing project setup or dependencies
* FOLLOW clean component-based architecture
* UI MUST be clean, modern, minimal, and premium-looking
* PRIORITIZE RESPONSIVENESS (mobile-first design)

---

DESIGN SYSTEM (VERY IMPORTANT):

THEME:

* Light theme ONLY
* Clean, soft UI (no harsh colors)

COLORS:

* Primary: Red (#ef4444)
* Accent: Orange (#f97316)
* Background: #f9fafb (very light gray)
* Card: white (#ffffff)
* Text:

  * Primary: #111827 (dark gray)
  * Secondary: #6b7280 (muted gray)

STYLE:

* Rounded corners: rounded-2xl
* Shadows: soft shadows (shadow-md / shadow-lg)
* Spacing: generous padding (p-4, p-6)
* Use Tailwind utility classes consistently

TYPOGRAPHY:

* Use Geist font (already installed)
* Headings: bold, clean
* Body: medium/light
